QR Code Generator (PyQt5 + Python)
A simple desktop application built with **PyQt5** and **qrcode** library that lets you generate and save QR codes for any text or URL.

Features:

* Enter text or URL to generate a QR code
* Select QR code **size** and **error correction level** (quality)
* Preview the generated QR code instantly
* Save the QR code as a PNG image

Requirements:

* Python 3.x
* PyQt5
* qrcode
* pillow (automatically installed with qrcode)

How to Run:

1. Install dependencies:
   pip install pyqt5 qrcode[pil]
2. Run the app:
   python qr_generator.py
